Rules to Play Jeopardy!

  1. Game Setup:
    • The game board contains 5 categories
    • Each category has 5 questions, valued from 200 to 1000 points, going by 200s
    • Higher point values = more difficult questions
  2. How to Play:
    1. Choosing a Question:
      • On your turn, select any available tile on the board (e.g., “History for 400”)
      • The question will appear in full-screen
    2. Submitting Your Answer:
      • Enter your answer into the submission form
      • Correct Answer: you earn the point value
      • Wrong Answer: no points (no penalty)
    3. Next Player’s Turn:
      • In two-player mode, the game alternates questions betweeen players
      • In four-player mode, the game alternates between teams, and player turn alternates between team turns.
  3. Answer Rules:
    • Each tile can only be selected once
    • No partial credit is given
    • Answers must be spelled correctly
    • Answers must be phrased as a response is optional depending on implementation (e.g., “What is the Eiffel Tower?”)
  4. Winning the Game:
    • The game ends when all 25 tiles have been answered
    • The player with the highest score wins
    • In the event of a tie, a sudden-death question may be used
